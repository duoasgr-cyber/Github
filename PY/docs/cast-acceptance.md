# 投屏录制功能 — 手动验收清单

## 前置条件

- [ ] Android 设备已通过 USB 连接且 `adb devices` 可见
- [ ] `lib/scrcpy-server.jar` 存在
- [ ] 工作流的 `device_resolution` 与设备实际分辨率一致

## 已知限制

1. **不支持多指手势**：当前仅录制单指 tap/swipe
2. **不支持长按**：长按超过 swipe 阈值距离时记录为 swipe，原地长按记录为 tap
3. **不支持按键事件**：Home/Back 等物理按键不在投屏窗口中，需手动添加
4. **坐标精度**：adb 降级模式下坐标经二次转换，可能有 ±1 像素误差
5. **画面旋转**：设备旋转后视频尺寸变化，control 通道需重新同步 video_size
