## 关键注意事项

### 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| EasyOCR 模型加载慢 | 首次启动需 10-30 秒 | 启动时异步加载，显示加载进度条 |
| ADB 连接不稳定 | 操作失败 | 每步执行前检查设备连接，失败自动重试 |
| PyQt5 线程安全 | GUI 卡顿或崩溃 | 所有 ADB 操作在子线程执行，通过信号槽更新 UI |
| JSON 配置损坏 | 程序无法启动 | 启动时校验 JSON 格式，损坏时自动恢复默认值 |
| 截屏选点坐标偏差 | 点击位置不准 | 严格按缩放比例换算，显示实际像素坐标供确认 |

### 兼容性注意

| 项目 | 注意点 |
|------|--------|
| Python 版本 | 建议 3.9+，EasyOCR 和 PyQt5 的兼容性最佳 |
| Windows 版本 | Win10/11，ADB 需在 PATH 中 |
| 手机分辨率 | 所有坐标基于特定分辨率，换设备需重新校准 |
| 游戏更新 | 游戏界面变化可能导致模板图片失效，需更新 tp/ 目录 |

### 开发规范

1. **线程安全**：所有 ADB 操作必须在子线程执行，通过 PyQt5 信号槽更新 UI
2. **EasyOCR 懒加载**：首次启动异步加载模型，显示进度条
3. **ADB 重连机制**：每步执行前检查设备连接，失败自动重试
4. **JSON 配置校验**：启动时校验格式，损坏时自动恢复默认值
5. **坐标缩放换算**：截屏选点时严格按缩放比例换算实际坐标
